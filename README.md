# Setup 

## clone this repo 

`git clone https://github.com/rapo7/ML-Assignment-7.git`

## create a virtualenv 

`python3 -m venv venv`
## Activate the venv 
`source venv/bin/activate`
or for windows 
`venv\Scripts\activate`

## Install all the requirements

`pip install -r requirements.txt`

## Run the project
`streamlit run app.py`
