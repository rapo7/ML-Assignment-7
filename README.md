# Setup 

## clone this repo 

```sh
git clone https://github.com/rapo7/ML-Assignment-7.git
```

## create a virtualenv 

```sh
python3 -m venv venv`
```
## Activate the venv 
```sh
source venv/bin/activate`
```
or for windows 
```cmd
venv\Scripts\activate
```

## Install all the requirements

```sh
pip install -r requirements.txt
```

## Run the project

```sh
streamlit run app.py
```
